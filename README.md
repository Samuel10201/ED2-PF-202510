# ED2 PF 202510
## To clone this repo use:
```
git clone https://github.com/sdariza/ED2-PF-202510.git
```
## To run docker use:
```
docker compose up
```
## To Stop and remove the containers created by docker compose up:
```
docker compose down -v
```
